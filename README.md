# 凯利公式头寸计算器

一个基于凯利公式的头寸管理工具，单文件纯前端实现，无任何外部依赖。

**[→ 在线使用](https://katarism.github.io/kelly/kelly.html)**

---

## 功能

### 实时计算
输入胜率、盈利比例、亏损比例，即时输出：
- **凯利头寸比例**（全凯利 / 半凯利 / ¼ 凯利）
- **本次投入上限**
- **本金预计余额**

公式：`f* = (p×b − (1−p)×a) / (a×b)`

### 交易记录 & 本金追踪
- 每轮选择策略（全 / 半 / ¼ 凯利或自定义金额）并锁仓
- 填写**本轮回收金额**，工具自动计算损益并更新本金，开启下一轮
- 完整历史记录，含投入金额、回收金额、损益、结束本金
- 底部汇总：总投资金额、总回收金额、总损益、损益占原本金百分比
- 数据通过 localStorage 持久化，刷新不丢失

### 资产曲线对比
交易记录达到 1 轮后自动展示折线图，对比：
- 实际资产走势
- 全 / 半 / ¼ 凯利理论资产走势

### 凯利之丘（风险收益全景图）
双轴图表，横轴为仓位大小，展示：
- **复合增长率**随仓位的变化曲线（左轴）
- **-50% 腰斩概率**随仓位的变化曲线（右轴，基于 Cramér-Lundberg 定理精确计算）
- ¼K / ½K / K 三个标记点的效能比评分

### 风险收益效能比
对三个策略点量化打分：

`效能比 = (本档增长率 ÷ 最大增长率) × (1 − 腰斩概率)`

数学结论：半凯利通常效能比最高（~65%），全凯利效能比固定为 50%（腰斩概率恒为 50%，数学定律）。

## 使用

直接打开 `kelly.html`，无需安装任何依赖，无需联网。

```
git clone https://github.com/katarism/kelly.git
open kelly/kelly.html
```

---

## 边界处理

| 情况 | 处理方式 |
|------|---------|
| 凯利值为负 | 显示"不建议投入" |
| 凯利值 > 100% | 显示上限 100% 并警告 |
| 亏损比例输入 > 100% | 自动截断至 100% |
| 回收金额输入为负 | 自动截断至 0 |
